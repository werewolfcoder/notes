
#### Set

A **set** is a well-defined collection of distinct objects called **elements** or **members**.

**Notation:** A = {1, 2, 3, 4} or A = {x | x is a natural number}



#### Subset

Set A is a **subset** of set B if **every element of A is also an element of B**.

**Notation:** A ⊆ B



#### Complement

The **complement** of set A (w.r.t. universal set U) is the set of all elements in U that are **NOT** in A.

**Notation:** A' or Aᶜ or Ā



|Property|Condition|Every element of B...|
|---|---|---|
|One-to-One|No two inputs → same output|may not be covered|
|Onto|Every output has an input|is covered (≥1 pre-image)|
|Bijection|Both one-to-one + onto|covered by exactly 1 pre-image|
|Inverse|Only for bijections|maps back to exactly 1 element of A|
