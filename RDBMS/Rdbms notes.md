
## Data:
	 data refers to the raw, unorganized facts, figures, or symbols that are stored      and managed within the database system.


## Information:

	Information is data that has been **processed, organized, and structured** to        make it meaningful and useful


## Database:

	A **structured collection of logically related data** designed to meet the          information needs of an organization.


## DBMS:
	A **software system** that enables users to define, create, maintain, and           control access to the database.

## Field:

	A field is a column in a database table that stores a specific type of data for all records.

## Record:
A record is a single, complete entry in a table, representing one instance of the entity.

## Metadata:
	It describes the **characteristics and structure** of the data


## Schema:
	A **logical design** or blueprint of the database


## Sub Schema:

Also known as the **external view** or **view level**. It defines the portion of the database that a particular user group or application program is authorized to access and interact with.


## Instance:
	The **actual content** of the database at any given point in time.



# Data dictionary

### 1. Definition

In simple terms, if your database is a library of books, the **Data Dictionary** is the card catalog. It doesn't contain the stories inside the books, but it tells you:

- What books exist (Table names)
    
- Where they are located (Storage details)
    
- Who wrote them (Creator/Owner)
    
- What rules they follow (Constraints)
    

**Formal Definition:** A centralized repository of information about data such as meaning, relationships to other data, origin, usage, and format.







# Data Types:
	A **Data Type** defines the kind of data that a column can hold, such as numeric, character, date/time, or large objects.


#### List of Common SQL Data Types (Examples)

1. **Numeric Types:** `INT` (or `INTEGER`), `DECIMAL`, `FLOAT`, `NUMERIC`.
    
2. **Character Types:** `CHAR` (fixed length), `VARCHAR` (variable length), `TEXT`.
    
3. **Date/Time Types:** `DATE`, `TIME`, `DATETIME`, `TIMESTAMP`.
    
4. **Large Object Types:** `BLOB` (Binary Large Object), `CLOB` (Character Large Object).
